# 🧬 Codon Translator Web App

A simple and interactive web application that translates DNA or RNA sequences into their corresponding amino acid chains using the standard genetic code.

---

## 🚀 Features

* 🔬 Translate DNA sequences into amino acids
* 🧪 Supports RNA codon input
* ⚡ Instant translation output
* 🧠 Uses standard genetic code table
* 💻 Clean and beginner-friendly UI

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## 🔧 How It Works

1. Enter a DNA or RNA sequence (e.g., `AUGGCUUAA`)
2. The app splits the sequence into codons (groups of 3)
3. Each codon is mapped to its corresponding amino acid
4. Output is displayed as an amino acid chain
