What is a Codon?

A codon is a triplet of nucleotide bases (e.g. AUG) in mRNA that encodes a specific amino acid. The genetic code maps all 64 codons to 20 amino acids.

Translation Process

Ribosomes read mRNA codons 5'→3'. tRNA brings matching amino acids. A polypeptide chain grows until a stop codon (UAA, UAG, UGA) terminates synthesis.

The Genetic Code

Near-universal across life. AUG is the start codon (Met). The code is redundant — multiple codons often encode the same amino acid (degeneracy).
